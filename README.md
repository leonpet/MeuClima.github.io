Passo 3: Criar um Repositório no GitHub
Faça login no GitHub

Clique no ícone "+" no canto superior direito

Selecione "New repository"

Preencha os dados:

Repository name: seu-usuario.github.io (CRÍTICO: deve ser exatamente assim)

Description: (opcional) Descrição do seu site

Selecione "Public"

Marque "Add a README file" (opcional, mas recomendado)

Clique em "Create repository"

# Faça o commit das alterações

# Envie para o GitHub
git push origin main



Passo 7: Ativar o GitHub Pages
No GitHub, vá para seu repositório

Clique em "Settings" (ícone de engrenagem)

No menu lateral, clique em "Pages"

Em "Branch", selecione "main" e a pasta "/ (root)"

Clique em "Save"
